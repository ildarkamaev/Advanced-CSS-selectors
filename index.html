> Direct Children

You can create selectors like this:

.direct-children p

That will select all paragraphs inside of .direct-children

But .direct-child > p will not select this paragraph. Because it's a grandchild to original element (original one has inside another element which has paragraph on its own). It will select only it's direct children.
